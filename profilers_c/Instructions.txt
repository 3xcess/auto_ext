Manually compile a profiler using this:
        clang -O2 -g -target bpf -D__TARGET_ARCH_x86 -c ba_bawm_{PROFILER}_skel.bpf.c -o ba_bawm_{PROFILER}_skel.bpf.o
        bpftool gen skeleton ba_bawm_{PROFILER}_skel.bpf.o > ba_bawm_{PROFILER}.skel.h
        gcc -O2 -g -o ba_bawm_{PROFILER} ba_bawm_{PROFILER}.c -lbpf


Run a profiler using this:
        sudo ./ba_bawm_{PROFILER}


Using the makefile:
        1. Create a new directory with the name of your profiler in uppercase\
           and add it to the Profilers section of the makefile

        2. Ensure the two files exist with profiler name in lowercase:
                ba_bawm_{PROFILER}.c
                ba_bawm_{PROFILER}_skel.bpf.c

        3. make all

        4. Check that ba_bawm_{PROFILER}.skel.h is generated and is not empty

        5. use the launch script to run the profilers


If vmlinux is missing:
        bpftool btf dump file /sys/kernel/btf/vmlinux format c > vmlinux.h


Map is available at:
        sudo bpftool map dump pinned /sys/fs/bpf/ba_bawm


Clear the map using
        sudo rm /sys/fs/bpf/ba_bawm
